Project Overview

Warriors of Nova Thera is a fast-paced roguelike featuring Pixelmon characters. As Lead Unity Programmer, I oversaw the technical architecture and core development through a successful Closed Beta release on Steam. My focus was on creating deep progression systems and a highly replayable, procedurally generated world.

Key Technical Contributions

  • Multiplayer Architecture: Integrated Photon Fusion to provide a high-performance, synchronized multiplayer experience.
  • Procedural World Building: Developed a custom hexagonal random map generation system, creating unique layouts for every run.
  • Progression & Skill Systems: Engineered a complex player growth framework featuring skill upgrades, level-based rarity scaling, and persistent ability unlocks.
  • Physics & Gameplay: Designed character movement, physics-based interactions, and combat loops to ensure a responsive, "game-feel" focused experience.
  • Dynamic Balancing: Built an intelligent enemy spawn and reward system that scales difficulty and loot based on real-time player progression.

Leadership & Optimization

  • Technical Oversight: Directed the development of Enemy AI and UI systems, ensuring they integrated seamlessly with the core gameplay architecture.
  • Performance Engineering: Optimized the game for Steam’s hardware standards, balancing high-fidelity assets with the performance demands of procedural generation and multiplayer.

Emergents TCG

Lead Unity Programmer (DoubleCoconut)

Project Overview

Emergents TCG was a high-performance, multiplayer turn-based card game built in Unity 3D. As Lead Programmer, I spearheaded a two-year development cycle from conception through a successful Open Beta release. While the project was later cancelled for non-technical reasons, the architecture remained rock-solid, supporting cross-platform play and competitive integrity.

Technical Contributions

  • Network Architecture: Implemented Photon PUN 2 for real-time multiplayer synchronization.
  • Security & Fair Play: Engineered a server-authoritative rules engine executed on a relay server to prevent client-side cheating and ensure gameplay consistency.
  • Cross-Platform Engineering: Optimized performance and managed Build Automation for a simultaneous release on macOS, Windows, Android, and WebGL.
  • Core Systems: Developed high-fidelity UI systems, core gameplay loops, and state-management frameworks tailored for complex TCG mechanics.

Leadership & Challenges

  • Technical Orchestration: Coordinated technical requirements and API integrations across five distinct teams: Unity, Rules Engine, Art, Backend, and Design.
  • Agile Delivery: Established a rapid deployment pipeline, consistently delivering stable weekly builds to stakeholders and testers.
  • Platform Optimization: Overcame significant performance hurdles to maintain a unified experience across mobile and web-based browsers.
